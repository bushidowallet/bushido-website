Bushido is an Open Source initiative around Bitcoin security server-side. From the functional perspective it is an implementation of Bitcoin wallet, as defined in BIP32 spec. It consists of an Android wallet application, a web-application built with AngularJS, service implemented in Java and Spring Boot. The main goal of this project is to provide a Bitcoin hot-wallet framework with highest possible security. This goal has to be achieved thanks to partnering with Utimaco Gmbh - a worldwide supplier of professional cybersecurity solutions. Bushido is leveraging their CryptoServer product line in order to secure sensitive data and cryptographic operations.
